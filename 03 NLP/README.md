# DataScience Projects
## 03 Обучение модели классификации комментариев / Определение токсичности комментариев.

Данный проект был выполнен в ходе обучения в Яндекс.Практикуме, профессии "Аналитик данных" / "Специалист по Data Science".

| Название проекта | Описание |Направление |Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- | :---------------------- |
| [Определение токсичности комментариев |Интернет-магазин запускает новый сервис. Теперь пользователи могут редактировать и дополнять описания товаров, как в вики-сообществах. То есть клиенты предлагают свои правки и комментируют изменения других. Требуется инструмент, который будет искать токсичные комментарии и отправлять их на модерацию..|Машинное обучение, NLP| *Python, Pandas, BERT, NLTK* |
